file: sample

Вызов метод mergePdf(first param, second param)
first param - Действия над файлами передаются в метод:
	- 'download' после склеивания сохраняется на стороне пользователя
	- 'browser' после склеивания выводит в браузер
	- 'file' после склеивания сохраняется на стороне сервера
	- default вывод в браузер
second param - директория в которую будет сохранено файл

file: main

Метод mergePdf

Параметры метода addPDF(dir, param)
param - При необходимость можно встроить в цикл исключения по обработке исходных файлов,
при этом с полученного пдф можно взять не все страницы, а любые существующие

При последовательном выборе страниц указывается так '1-3'
Если исключительно определенные то '1,5,9'

default: 'all' - все страницы
